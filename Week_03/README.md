# Week 3 — Functions, Scope & Recursion

### Weekly Objective:

This week, I focused on learning how to use Functions to make my code organized and reusable. I also studied the LEGB Rule, which is a simple way to understand how Python finds and manages variables in different parts of a program. These basics are very important for writing clean code and building smart AI projects in the future.

### Featured Project: CLI Quiz Game
A simple command-line game that asks questions and tracks the score.
- **Features:** Uses functions to handle questions and keeps track of the player's score.
- **Concepts Applied:** Local and Global variables, returning values from functions, and organizing code into modules.

### Key Learning Insights:
- **Strawberry Tree Analogy:** I used a simple "tree" story to remember the difference between Local and Global variables.
- **Clean Code:** I learned that using functions makes it much easier to read and fix my code later.
- **Step-by-Step Search:** I now understand the exact order Python uses to look for a variable's name.
